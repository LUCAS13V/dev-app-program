lista ordenada

listas ordendas usam a tag ol

as listas ordendas usam a tag ol que vem do odernid listem ou lista ordernada

e os item delas sao marcados com a tag li que vem do item list ou item da lista

exemplo de lista ordenada

  1. item
  2. item
  3. item
  4. item

para fazer a lista comecar de um determinado nuemro coloque o parametro start="numero inicial"

lista cmc do numero 8
  1. item
  2. item
  3. item
  4. item

tipos de lista

type="1"

normal contagem numerica

type="A"

contagem em ordem alfabetica maiscula para deixar a letra minuscula basta escrever em minusculo type="a"

type="i"

contagem em nuemros romanos a mesma funcao de masicula ou minuscula tbm se aplica ao tipo i


listas nao ordenadas

listas nao ordenadas usam a tag ul

a tag ul vem de underned listem ou lista desordenada
